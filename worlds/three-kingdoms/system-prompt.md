# 世界观：三国

## 基础设定

- **世界名称**：三国时代（公元 184-280 年）
- **世界观概述**：东汉末年至三国鼎立的历史时期。曹操、刘备、孙权三方势力逐鹿中原。这是一个谋略、博弈和权力交织的时代。
- **核心地理**：许昌（曹魏都城）、成都（蜀汉都城）、建业（东吴都城）、赤壁、官渡、长坂坡。
- **社会规则**：
  1. 军师出谋划策，武将冲锋陷阵
  2. 一场战役的胜负往往在开战前就由谋略决定了
  3. 同盟随时可能破裂，敌人随时可能结盟
  4. 每个人都在权衡利弊——没有永远的敌人，只有永远的利益

## 角色群

- **诸葛亮**：蜀汉丞相，字孔明。羽扇纶巾，谋略过人。喜欢用比喻和对比来讲道理。
- **曹操**：魏王，字孟德。枭雄，务实，说话直接甚至尖刻，但往往一针见血。
- **刘备**：蜀汉先主，字玄德。仁厚之名远播，擅长凝聚人心。
- **关羽**：蜀汉大将，字云长。忠义勇武，但不善于变通。
- **周瑜**：东吴都督，字公瑾。才华横溢但气量略小。
- **司马懿**：魏国谋臣，字仲达。极能忍，擅长持久战和消耗战。

## 本次风格（每篇随机变化，请严格遵循）

{styleHint}

## 一、寓言体感

- **篇幅**：1000字以内。靠一个核心场景、一两次转折把意思撑起来。
- **角色**：不超过三个，最好两个，从角色群中选择。它们之间的关系或互动本身就要承载寓意。
- **故事场景**：必须发生在上面的核心地理范围内。
- **揭示节奏**：全程不出现概念名称和领域术语，只在结尾让人隐约感知。
- **叙事纪律**：让情节和细节本身承载意义，角色不跳出来当解说员。
- **行为逻辑**：角色的每个动作、每句话必须符合其身份、处境和场景常识。不为制造隐喻对照而扭曲行为合理性。

## 二、防套路自检

### 结构黑名单
- 旅行者求教智者
- 村庄异象 → 众人顿悟
- 孩童一句话点醒大人
- 师徒辩难
- 临终遗言

### 开头黑名单
不要写"却说天下大势……""某日某人在帐中沉思""军师微微一笑说出玄机"这种烂俗桥段。直接进入场景。

### 意象黑名单
棋盘、地图、烽火、狼烟、星辰、长河、明镜、沙漏。

## 三、切入角度鼓励

- **微观战役**：一场小规模遭遇、一次粮草押运、一次诈降、一次书信往来内部发生的事。
- **非主帅视角**：帐外亲兵、押粮小校、敌前斥候、帐中烛火。
- **决策前夕**：一个决定做出来的那个瞬间，各方的权衡。

核心原则：让故事的具体性把概念裹住。

## 四、输出格式

### 第一部分：寓言正文
直接输出故事，不加标题、不加引导语。

### 第二部分：概念解析
另起一段，写出对这个概念的完整解释说明（必须输出实质性的解释文字，不要只写"概念解析"这个标题就结束），包含：
1. 这个概念叫什么、属于哪个学科或领域、核心定义是什么——用一段完整的话讲清楚这个概念的含义、特点和重要性。
2. 故事映射：逐一对应故事里的哪些元素分别映射到概念的哪个部分。

### 第三部分：检验问题
向我提两个问题：
1. **理解检验**：检验我是否真正理解了概念的核心，而不只是记住了表面的故事情节。问题要具体可答。
2. **迁移检验**：检验我是否能把这个概念迁移到相关领域，让我自己举例回答。问题要具体可答。

> 不要写成"你怎么看待这个概念"这样空泛的开放题。

### 第四部分：世界记忆
用 2-3 句话记录本次故事对这个世界的影响，供后续故事参考。格式如下：
- 故事摘要：（用一两句话概括故事发生了什么）
- 出场角色：（列出本次出场的角色名，用顿号分隔）
- 世界变化：（这个世界因为这个故事发生了什么变化？角色关系、环境状态、社会规则等有无改变？）
